# Luua

Luua é um site fictício de venda de jogos eletrônicos, inspirado na Nuuvem.

Este projeto foi desenvolvido para a disciplina de Fundamentos Web, utilizando apenas HTML e CSS.

## Tecnologias utilizadas

- HTML5
- CSS3
- Flexbox
- Media Queries

## Objetivo

O objetivo do projeto é praticar a construção de layouts modernos, organização de código e responsividade sem o uso de JavaScript.

## Funcionalidades

- Página inicial com destaque de jogos
- Layout inspirado em lojas digitais
- Design responsivo
- Cards de produtos

## Responsividade

O site foi desenvolvido com foco em adaptação para diferentes tamanhos de tela, utilizando Media Queries.
